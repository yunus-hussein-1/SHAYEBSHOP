# SHAYEB SHOP — متجر الشايب 🛍️

متجر إلكتروني كامل بثلاث لغات (عربي / English / Türkçe) — واجهة + خادم خلفي + قاعدة بيانات.

## 📁 هيكل المشروع

```
shayeb-project/
├── index.html            ← هيكل الصفحة (HTML)
├── css/
│   └── style.css         ← التصميم كامل (الألوان، البطاقات، السلة...)
├── js/
│   ├── config.js         ← ⚙️ إعدادات المالك: إيميلك، غوغل، العملات، الكوبونات
│   ├── i18n.js           ← الترجمات الثلاث كاملة
│   ├── data.js           ← الفئات والقوائم والمنتجات
│   └── app.js            ← منطق التطبيق: دخول، سلة، دفع، إدارة، بائع
├── backend/
│   ├── server.js         ← الخادم الخلفي (API) — Node.js + Express
│   ├── db.js             ← الاتصال بقاعدة البيانات وتجهيزها
│   ├── database.sql      ← مخطط قاعدة البيانات كامل (11 جدول)
│   ├── package.json      ← تعريف المشروع والمكتبات
│   └── .env.example      ← ملف الإعدادات السرية (انسخه باسم .env)
└── README.md             ← هالملف
```

## 🖥️ تشغيل الواجهة (هلق فوراً)

افتح `index.html` بالمتصفح — بتشتغل مباشرة (بوضع تجريبي، البيانات ضمن الجلسة).
أو ارفع المشروع على GitHub Pages / Netlify.

**حسابات التجربة:** زبون `demo@shayeb.shop`/`123456` — إدارة `owner@shayeb.shop`/`admin123`

## 🚀 تشغيل الخادم الخلفي (لما يجهز السيرفر السحابي)

يحتاج تنصيب Node.js (نسخة 18+) من nodejs.org ثم:

```bash
cd backend
cp .env.example .env      # وعدّل القيم: إيميلك، كلمة سر الإدارة، JWT_SECRET
npm install               # تنزيل المكتبات
npm start                 # 🚀 API شغّال على http://localhost:3000
```

قاعدة البيانات (ملف `shayeb.db`) بتنشأ تلقائياً أول تشغيل مع حساب المدير من `.env`.

## 🔌 الـ API الجاهز (أهم المسارات)

| المسار | الوظيفة |
|---|---|
| POST /api/auth/register · login · forgot · reset | الحسابات + استعادة كلمة السر بكود |
| GET/PUT /api/me · /password | الملف الشخصي + تغيير كلمة المرور + حذف الحساب |
| GET/POST /api/products · comments | المنتجات + التعليقات والتقييم |
| POST /api/orders · delivered · cancel · rate | الطلبات + شام كاش + الإلغاء + التقييم |
| POST /api/seller/apply · posts · earnings | طلب البيع (شام كاش إلزامي) + المنشورات + الأرباح |
| /api/admin/... | لوحة الإدارة: قبول/رفض، حظر، شكاوى، بلاغات |

💰 **منطق الأرباح مبرمج بالخادم:** عند تأكيد الاستلام، بينشأ سجل تحويل تلقائي للبائع
باستحقاق **بعد 7 أيام** وبخصم **عمولة 10%** (جدول payouts).

## ⚙️ قبل الانطلاق — عدّل:

1. `js/config.js` ← ADMIN_EMAIL + ADMIN_PASS + GOOGLE_CLIENT_ID (تعليمات غوغل فوق السطر 🔑) + أسعار الصرف
2. `backend/.env` ← نفس الإيميل + JWT_SECRET عشوائي طويل

## 🔗 الخطوة الجاية (منعملها سوا وقت السيرفر)

ربط الواجهة بالـ API: تحويل دوال `app.js` من التخزين المؤقت إلى نداءات `fetch` للخادم —
المسارات جاهزة ومتطابقة مع ميزات الواجهة وحدة بوحدة.
